# Changelog

All notable changes to VDAS-Insight are documented here.
This project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] — 2026-07-22

初回リリース。大規模車両計測データ（J1939 等）の前処理済み parquet / csv を対象とした、
研究・業務向けの可視化・統計・コホート比較デスクトップアプリ。

### Added
- **解析エンジン (`vdas/`, UI 非依存)**
  - スキーマ非依存の列 *役割* マッピング（時間 / ギア / フラグ / 車速 / 数値 / カテゴリ）と自動推定
    （車速とエンジン回転を区別）。
  - ギア段分析：変速検出（アップ/ダウン）、変速回数・時間あたり・距離あたり、ギア段別滞在時間、
    変速の遷移行列。
  - フラグ (0/1) 分析：立上り回数、ON 率、ON 時間分布、立上り間隔分布。
  - **派生信号**：車速 → 加速度 (m/s²)・加加速度 (m/s³)、汎用の微分/2階微分/移動平均/絶対値。連鎖可能。
  - **タグ / コホート**：タグに **カテゴリ**（メーカー・走行条件など）を付与、多対多、
    タグ AND 絞り込み（`dataset_ids_matching_all`）。
  - **コホート比較**：示量量をプールしてレートを再計算する公平比較、データ単位のばらつき、
    基準=100 の相対指数。**タグごと** / **カテゴリで分割** の 2 モード＋ AND 絞り込み
    （例:「高速に絞ってメーカー同士を比較」）。
  - `prepare()` の自己無効化キャッシュ（ファイル更新時刻＋役割＋派生信号の定義をキー）。
- **デスクトップ UI (`desktop/`, PySide6 + pyqtgraph)**
  - INCA / CANape 風のライトテーマ・ドッキング可能パネル（データセット / 信号ツリー / 役割 / タグ）。
  - タブ：計測(時系列・リンク軸・共有クロスヘア)、ギア段、フラグ、統計、コホート比較。
  - 派生信号の作成ダイアログ、CSV エクスポート。
  - **データ管理**画面（複数選択でタグの一括付与/解除・一括削除）、読み込み時の一括タグ付与、
    タグのカテゴリ編集。
- **永続化**：DuckDB がクエリエンジンとメタデータストアを兼任（datasets / column_roles / tags /
  dataset_tags / derived_signals）。
- **サンプルデータ生成**（`scripts/gen_sample.py`）と、解析エンジン + デスクトップ起動スモークの
  テスト一式。
